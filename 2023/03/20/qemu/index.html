qemu的安装

1.安装中遇到的问题

出现E: Failed to fetch等问题，网上查了一下应该换源

首先进行备份

sudo cp /etc/apt/sources.list /etc/apt/sources.list.old

编辑sources.list文件

sudo vim /etc/apt/sources.list

将原来的内容覆盖成新的镜像源

进行更新

sudo apt-get updata

2.qemu的安装

直接安装

sudo apt-get install qemu

接着

sudo apt-get install qemu-system

sudo apt install qemu-user
